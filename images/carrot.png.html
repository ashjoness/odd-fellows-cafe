To resize your logo in CSS without stretching or distorting it, you can control its size using either max-width, height, or width while keeping the aspect ratio intact. Here's how to do it:

css

✅ Option 2: Use both width and height but keep aspect ratio

css

If you want a quick fix in HTML:

html

css
